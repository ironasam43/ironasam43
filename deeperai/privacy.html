Privacy Policy — DeeperAI

Last updated: 2026-04-04

Information We Collect

DeeperAI does not collect, store, or transmit any personal information about its users.

Where Your Data Is Stored

  • Local storage: All data (themes, notes, tags, etc.) is stored on your device in a .dpai data file.
  • iCloud Drive (optional): If you save your data file to iCloud Drive, it is synced to your other devices via Apple's iCloud service. The developer has no access to this data.

Third-Party Data Transmission

When using the Chat or Antenna features, content you enter (theme titles, notes, etc.) is sent to Anthropic's API via Claude Code. Please refer to Anthropic's Privacy Policy for details on how that data is handled.

DeeperAI itself does not send data to any third party other than Anthropic.

Analytics & Advertising

DeeperAI does not use any analytics tools or advertising SDKs.

プライバシーポリシー — DeeperAI

最終更新：2026-04-04

収集する情報

DeeperAI は、ユーザーの個人情報を収集・保存・送信しません。

データの保存場所

  • ローカル保存：すべてのデータ（テーマ・メモ・タグ等）はデバイス上の .dpai データファイルに保存されます。
  • iCloud Drive（任意）：データファイルを iCloud Drive に保存した場合、Apple の iCloud サービスを通じてご自身の他のデバイスと同期されます。開発者はこのデータにアクセスできません。

第三者サービスへのデータ送信

チャット機能・アンテナ機能を使用する場合、入力内容（テーマのタイトル・メモ等）が Claude Code を通じて Anthropic の API に送信されます。送信されるデータの取り扱いについては、Anthropic のプライバシーポリシーをご確認ください。

DeeperAI 自体は、Anthropic 以外の第三者にデータを送信しません。

アナリティクス・広告

DeeperAI はアナリティクスツールや広告 SDK を使用しません。
